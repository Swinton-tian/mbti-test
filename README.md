# MBTI 人格测试网站

这是一个简单的 MBTI（Myers-Briggs Type Indicator）人格测试网站，使用 HTML、CSS 和 JavaScript 构建。

## 如何运行

1. 打开 `index.html` 文件在浏览器中。
2. 回答所有问题，选择最符合你的选项。
3. 点击“提交并查看结果”按钮。
4. 查看你的 MBTI 类型和简短描述。

## 文件结构

- `index.html`: 主页面
- `script.js`: JavaScript 逻辑
- `style.css`: CSS 样式
- `README.md`: 本说明文件

## 注意

这是一个简化的测试版本，仅供娱乐和教育目的。真正的 MBTI 测试需要更全面的问题和专业评估。
